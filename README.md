# Extension

### Description

Boilerplate project that can be used to bootstrap browser extensions.  

### Folder structure
```
src
├── css
├── js
│   ├── content-script.js
│   ├── popup.js
|   ├── service-worker.js
|   └── common
|       ├── constants.js
|       ├── context.js
│       └── functions.js
├── config.sample.js
├── manifest.json
└── popup.html
```

### Config

Copy config.sample.js as config.js.

| Field | Type | Description | Default |
| ----- | ---- | ----------- | ------- |
